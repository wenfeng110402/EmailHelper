1）基础标签：页面的骨架 入门

了解最常见的块级与行内元素，以及结构的最小单位。

<!DOCTYPE html> <html lang="zh-CN"> <head> <meta charset="UTF-8"> <title>我的页面</title> </head> <body> <h1>一级标题</h1> <p>这是一个段落。</p> 行内元素：<span>不会换行</span> </body> </html>
  • 块级元素（如 div, h1, p）独占一行。
  • 行内元素（如 span, a）不换行，常用于修饰文字。
  • 语义化：用合适标签表达含义，例如标题用 h1-h6，段落用 p。

2）图片与链接 入门

学会放置图片、编写可点击的链接，并添加替代文本。

<img src="https://via.placeholder.com/200x80" alt="占位图"> <a href="https://example.com" target="_blank">访问网站</a>

alt 用于图片无法显示时的替代文字，同时提升无障碍与 SEO。

3）结构与语义 基础

合理的页面结构提升可读性与可维护性。

<header>站点头部（Logo/导航）</header> <main>主要内容</main> <footer>底部信息</footer>
  • header/main/footer 提供更清晰的语义结构。
  • 为不同区域设置独立样式更容易。

4）CSS选择器与盒模型 基础

掌握选择器语法与元素的内容区、内边距、边框、外边距。

/* 选择器 */ .card { padding: 16px; margin: 12px 0; } .card h2 { color: #1d1d1f; } #unique { border: 2px solid #d2d2d7; } /* 盒模型 */ .box { width: 200px; padding: 12px; border: 1px solid #ccc; margin: 8px; }

常用选择器：类（.class）、ID（#id）、标签（div）。优先级：ID > 类 > 标签。

5）现代布局：Flex 与 Grid 进阶

用现代布局快速实现响应式排版。

Flex 弹性布局

.row { display:flex; gap:12px; } .row .item { flex: 1; padding: 12px; border:1px solid #ddd; }
父元素 display:flex 后，子元素可通过 flex 控制伸缩。

Grid 网格布局

.grid { display:grid; grid-template-columns: repeat(3, 1fr); gap:12px; } .grid .cell { padding: 12px; border:1px solid #ddd; }
Grid 适合二维网格，repeat(列数, 1fr) 可均分宽度。

6）过渡与动画 进阶

让交互更顺滑，自然提升体验。

.btn { background:#0b66ff; color:#fff; padding:8px 14px; border-radius:8px; transition: transform .2s ease; } .btn:hover { transform: translateY(-2px); }

过渡 transition 常与 :hover 搭配，注意时序和缓动函数。
